\doxysubsection{Detailed Description}
Klasa do obsługi API GIOŚ -\/ Głównego Inspektoratu Ochrony Środowiska. 

Klasa zapewnia metody do pobierania danych o jakości powietrza z API GIOŚ, w tym stacje pomiarowe, czujniki, pomiary oraz indeksy jakości powietrza.
